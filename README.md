# Reference Object Detection

A query-guided object detection system using YOLOv8 and CLIP.

## Project Idea

The user uploads:

1. A reference object image
2. A target scene image

The system identifies and highlights matching objects in the target image.

Example:

* Upload an apple image
* Upload a fruit basket image
* System highlights all apples in the basket

## Tech Stack

* Python
* YOLOv8
* CLIP
* PyTorch
* OpenCV
* FastAPI
